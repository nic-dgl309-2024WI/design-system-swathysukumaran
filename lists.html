Lists

Lists in HTML are like grocery lists or to-do lists. They help organize information by presenting it in a structured way. You can have numbered lists (like steps in a recipe) or bulleted lists (like items on a shopping list). Each item in the list is wrapped in

  • tags, which stand for "list item".

    Code

                        <li>Item 1</li>

                        <li>Item 3</li>
                        </ul>
